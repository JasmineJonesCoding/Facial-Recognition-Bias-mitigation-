# Facial Recognition Bias Mitigation Utilizing Machine Learning

## Overview

This project investigates racial bias in facial recognition systems using the UTKFace dataset.

A baseline Logistic Regression model was first trained and evaluated across various demographic groups. A group-weighted model was then implemented to improve performance for underrepresented groups within a dataset.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- KaggleHub

## Key Results

- Baseline Black-group accuracy: 79%
- Weighted-model Black-group accuracy: 90%
- Improved fairness through group-weighted training

UTKFace Dataset
